# spsmb-musya
Web sistem penerimaan santri dan murid baru Yayasan Mukhtar Syafa'at

## Deployment
Proyek ini dikonfigurasi untuk dideploy secara otomatis ke **GitHub Pages** setiap kali ada perubahan pada branch `main`.

### Konfigurasi Manual GitHub Pages:
1. Masuk ke tab **Settings** di repositori GitHub Anda.
2. Pilih menu **Pages** di sisi kiri.
3. Di bawah **Build and deployment**, pastikan **Source** diatur ke **GitHub Actions**.
